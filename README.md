# Onboarding for Brand

A bilingual (EN / 中文) portal for the *Onboarding for Brand v5* framework — a 42-question brand archetype audit by Alan Yin / EDUPUNK.

The framework itself is at [`public/brand-archetypes-v5.md`](public/brand-archetypes-v5.md), licensed under [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/).

## Stack

Next.js 16 · React 19 · Tailwind 4 · GSAP · Lenis · Drizzle + Neon Postgres · Vercel AI SDK + OpenRouter.

## Dev

```bash
bun install
bun dev
```

Open <http://localhost:3000>.
